# TrendWatch landing page

A single self-contained `index.html` (inline CSS, no build step, no dependencies)
for a TrendWatch landing page. It is designed to live on **your own domain** so
all SEO and brand authority stay on trendsmcp.ai.

## Recommended: host it on your domain
Drop it at **`trendsmcp.ai/trendwatch`** (or `/trendwatch/index.html`). One file, nothing to build.

## Alternative: GitHub Pages
If you ever want it on GitHub instead, enable Pages (Settings → Pages) and point it
at this folder, or move `index.html` to the repo root of a `gh-pages` branch. Note
this serves from `*.github.io`, which does **not** build authority on trendsmcp.ai —
hosting on your own domain is preferred.
